# 跟雪婷老师学汉语 — 学習アプリ README

**静岡大学 人文社会科学部 中国語学研究室 漢使研**  
キャンパス・ストーリー 第2版｜1学期（30回 / 15週）対応

---

## 概要

日本人大学生（＋ベトナム人・韓国人留学生）約50名のクラス向けに設計された中国語自習補助アプリです。授業では説明が多くなりがちで個別練習の時間が取りにくいため、アプリで口頭練習・発音フィードバック・文化体験をカバーします。

> 「無理せず、楽しみながら授業とともに中国語を学ぶ」

## 画面構成（5タブ）

### 📚 授業タブ
- 1学期カリキュラム一覧（発音編＋第1〜12課）
- 週ごとにグループ表示
- 各課の文法ポイント・語彙数をひと目で確認

### 💬 練習タブ
各課を選択して4つのモードで学習：

| モード | 内容 |
|---|---|
| 本文 | 課文をセンテンスごとに表示。ピンイン・日本語訳付き。TTS発音ボタン |
| 単語 | フラッシュカード形式。全単語リスト付き |
| 口頭練習 | 授業スライドの「想一想 答一答」をそのまま収録。タップで模範解答表示 |
| AI会話 | 雪婷老師役のAIとテキスト会話。各課のテーマに沿ったシチュエーション |

### 🎤 発音タブ（平平 piángpiàng コーナー）
静岡大学漢使研マスコット「平平 piángpiàng（唐装パンダ）」が3つのモードで案内：

| モード | 内容 |
|---|---|
| フレーズ | 6カテゴリ（声調・変調・早口言葉・各課フレーズ・歌）のドリル。マイクで録音→AIが5段階評価＋改善点をピンインで指摘 |
| 会話 | 平平とフリー会話。音声入力またはテキスト入力。発音・声調・文法ミスを優しく指摘 |
| 文化 | 15回中10課分の中国文化コンテンツ（下記参照） |

### ✏️ テストタブ
- 各課10問・10点満点（教員の成績評価対応）
- 選択式・意味問題・空欄補充・文型選択など多様な形式
- 即時フィードバック（正解・不正解の色分け）
- 正答率と点数をリアルタイム表示

### 📖 読解タブ
漢使研オリジナル阅读文「毛毛の大学生活」を教材化：
- 重要語彙16語をハイライト表示 → タップで意味ポップアップ＋TTS
- 唐詩《春晓》コーナー（ピンイン・日本語訳・TTS朗読）
- 読解問題8問（タップで解答）
- AI雪婷老師への質問チャット

## AI機能の仕組み

Anthropic Claude API（`claude-sonnet-4-20250514`）を使用。

| 機能 | モデルへの指示 |
|---|---|
| AI会話（練習タブ） | 雪婷老師として各課のシチュエーションで会話・文法指導 |
| 発音評価（フレーズモード） | 5段階★評価・よかった点・改善ポイントをピンインで指摘 |
| 平平との会話 | 平平 piángpiàng として発音・声調・文法ミスを優しく修正 |
| 読解チャット | 阅读文の内容・語彙・文法について質問に回答 |

> **注意：** Claude.aiのアーティファクト環境ではAPIキーが自動注入されます。外部環境に持ち出す場合は `fetch()` のヘッダーに `x-api-key` を追加してください。

## 開発ノート

### 音声入力について
- ブラウザの Web Speech API（`SpeechRecognition`）を使用
- 言語設定：`zh-CN`（普通話）
- Chrome のみ完全対応。非対応ブラウザではテキスト入力で全機能を利用可能
- 音声入力ボタンはChrome検出時のみ有効化

### ファイルサイズ
- `xue_ting_chinese_app.html`：約197KB（平平画像base64込み）
- `hanshiken_music.html`：約15MB（MP3音声3曲base64込み）

### 多言語対応の考え方
- 日本語・中国語・ピンインを常に3段表示
- 文化コンテンツに日中越韓の比較（两只老虎・方言など）
- テスト問題は日本語UIで出題、解答も日本語で確認可能
